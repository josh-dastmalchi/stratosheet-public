Views

Views allow you to customize how you see data from your tables. There are multiple kinds of views, which you can use to present subsets of your data or to visualize them in different ways.

Each table can have any number of views, and each view must have a unique name.

When a table is created, a default table view is automatically created, and cannot be removed.

There are three types of views:

  • Table
  • Calendar
  • Kanban
