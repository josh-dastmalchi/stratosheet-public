Automation Actions

Automation actions are the individual steps taken when executing an automation. They typically interact with external systems and can use data from the automated table to perform their tasks.

These actions take place in the order they are configured.

Slack

This action allows a Slack message be sent to a public channel in a Slack workspace.

Gmail

This action allows an email to be sent from a connected Google account.

Twilio

This action allows an SMS to be sent from a connected Twilio account's phone number.

Google Calendar

This action allows a Google Calendar event to be created from a connected Google account.
