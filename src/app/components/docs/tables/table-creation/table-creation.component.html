Creating a Table

How To

To create a table, you'll choose a name for the table, which must be unique in the workspace.

Optionally, you can choose to create it from reference data, which will automatically create columns for the table and pre-populate it with some rows. This reference data is for common things like months of the year, so that you won't have to input it manually.
